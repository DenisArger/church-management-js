# Команда /weekly_schedule

## Назначение

Получает и отправляет расписание служений на предстоящую неделю, которые требуют рассылки.

## Параметры

Нет

## Пример использования

```
/weekly_schedule
```

## Реализация

**Файл**: `src/commands/weeklyScheduleCommand.ts`

## Информация включает

- Период недели
- Служения по дням:
  - Время
  - Название
  - Описание
  - Место проведения
- Случайное благословение

## Режим DEBUG

В режиме DEBUG рассылка не отправляется, показывается сообщение о том, что режим активен.
