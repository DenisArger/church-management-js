# Команда /add_prayer

## Назначение

Добавляет молитвенную запись на текущую или следующую неделю.

## Параметры

- Без параметров - показывает справку
- С параметрами: `Имя | Тема | Неделя`

## Формат ввода

```
Имя | Тема | Неделя (current/next)
```

## Пример использования

```
/add_prayer
/add_prayer Иван Петров | Здоровье | current
/add_prayer Мария Сидорова | Работа | next
```

## Реализация

**Файл**: `src/commands/addPrayerCommand.ts`

## Валидация

- Имя: минимум 2 символа
- Тема: минимум 3 символа
- Неделя: `current` или `next`

## Недели

- `current` - текущая неделя (понедельник - воскресенье)
- `next` - следующая неделя

## Описание

Команда парсит ввод пользователя, валидирует данные, вычисляет даты недели и сохраняет запись в Notion. При успешном сохранении отправляет подтверждение с деталями записи.
