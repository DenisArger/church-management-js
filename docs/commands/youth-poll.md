# Автоматическое создание опросов для молодежных событий

## Назначение

Опросы о посещении молодежных встреч создаются автоматически через единый планировщик `poll-scheduler`. Он работает для событий типов «Молодежное» и «МОСТ».

## Автоматизация

Планировщик запускается **каждые 15 минут**.

## Реализация

**Файл**: `netlify/functions/poll-scheduler.ts`

## Логика

- Ищет события на ближайшие 48 часов
- Отправляет опрос **за 24 часа** до события
- Отправляет уведомление администратору **за 3 часа** до события
- Использует стандартную генерацию текста опроса
