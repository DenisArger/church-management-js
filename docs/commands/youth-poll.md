# Команда /youth_poll

## Назначение

Создает опрос о посещении молодежного служения на завтра, если такое событие запланировано.

## Параметры

Нет

## Пример использования

```
/youth_poll
```

## Реализация

**Файл**: `src/commands/youthPollCommand.ts`

## Функциональность

- Проверяет наличие молодежного события на завтра в календаре
- Извлекает тему и время из события
- Создает опрос в молодежной группе
- Поддерживает debug режим (отправка в тестовую группу)

## Опции опроса

- "Да, приду"
- "Нет, не смогу"
- "Пока не знаю"

## Описание

Команда проверяет календарь на наличие молодежного события на завтра. Если событие найдено, извлекает тему и время, затем создает опрос в соответствующей группе (production или debug).
