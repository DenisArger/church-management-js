# Команда /request_pray

## Назначение

Отправляет рассылку с информацией о людях, за которых нужно молиться, с датами последней молитвы.

## Параметры

- `date` или `дата` (опционально) - сортировка по дате (по умолчанию)
- `name` или `имя` (опционально) - сортировка по алфавиту

## Пример использования

```
/request_pray
/request_pray date
/request_pray name
```

## Реализация

**Файл**: `src/commands/prayerRequestCommand.ts`

## Функциональность

- Получает все записи из Notion
- Группирует по людям, находит последнюю дату молитвы
- Отправляет полный список (с указанной сортировкой)
- Отправляет список из 3 человек (сортировка по дате, старые первыми)

## Режим DEBUG

В режиме DEBUG рассылка не отправляется, показывается сообщение о том, что режим активен.
