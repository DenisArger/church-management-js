# Команда /daily_scripture

## Назначение

Получает и отправляет ежедневное чтение Библии из Notion.

## Параметры

Нет

## Пример использования

```
/daily_scripture
```

## Реализация

**Файл**: `src/commands/dailyScriptureCommand.ts`

## Данные

- Текст чтения
- Ссылка на стих
- Перевод

## Описание

Команда запрашивает из Notion базы данных ежедневное чтение для текущей даты, форматирует его и отправляет пользователю.
