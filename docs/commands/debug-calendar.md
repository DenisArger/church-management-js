# Команда /debug_calendar

## Назначение

Отладочная команда для проверки данных в календаре Notion.

## Параметры

Нет

## Пример использования

```
/debug_calendar
```

## Реализация

**Файл**: `src/commands/debugCalendarCommand.ts`

## Описание

Команда выводит отладочную информацию о записях в календаре Notion, включая доступные поля и их значения.
