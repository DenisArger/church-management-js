# Команда /create_poll

## Назначение

Создает простой опрос о посещении молодежной встречи.

## Параметры

Нет

## Пример использования

```
/create_poll
```

## Реализация

**Файл**: `src/commands/createPollCommand.ts`

## Опции опроса

- "Да, приду"
- "Нет, не смогу"
- "Пока не знаю"

## Описание

Команда создает опрос в текущем чате с вопросом "Приходите ли вы на молодежную встречу?" и тремя вариантами ответа.
