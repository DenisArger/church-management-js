# Команда /prayer_week

## Назначение

Показывает информацию о людях, за которых нужно молиться на следующую неделю, и темы молитв.

## Параметры

Нет

## Пример использования

```
/prayer_week
```

## Реализация

**Файл**: `src/commands/prayerWeekCommand.ts`

## Функциональность

- Вычисляет даты следующей недели (понедельник - воскресенье)
- Фильтрует записи, которые пересекаются с этим периодом
- Группирует по людям
- Показывает тему молитвы и период
