# Команда /test_notion

## Назначение

Проверяет подключение к Notion API и доступность баз данных.

## Параметры

Нет

## Пример использования

```
/test_notion
```

## Реализация

**Файл**: `src/commands/testNotionCommand.ts`

## Описание

Команда проверяет доступность всех настроенных баз данных Notion и выводит информацию о статусе подключения.
